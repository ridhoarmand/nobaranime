# NobarAnime 🎬

NobarAnime adalah platform *streaming* khusus anime yang dibangun untuk memberikan pengalaman menonton yang cepat, responsif, dan ringan. Beroperasi sebagai bagian dari ekosistem NobarFilm, aplikasi ini dirancang khusus untuk memenuhi kebutuhan *weaboo* dan pecinta anime dalam melacak jadwal tayang (Ongoing/Upcoming) serta menonton episode terbaru.

## Tech Stack
Aplikasi ini dikembangkan modern dan *scalable* menggunakan:
- **Frontend Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Data Fetching:** [TanStack Query](https://tanstack.com/query/latest)
- **Routing:** [React Router](https://reactrouter.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Auth & Database:** [Supabase](https://supabase.com/) (menggunakan skema *Cross-Domain SSO Cookies*)

## Ekosistem Nobar 🌐

NobarAnime hanyalah satu dari beberapa *sub-project* di bawah naungan ekosistem Nobar. Berikut status pengembangan platform lainnya:

- 🍿 **[NobarFilm](https://film.idho.eu.org)** — Platform utama untuk *streaming* film dan TV Series. **(Live)**
- 🐼 **NobarDracin** — Platform khusus Drama China / Donghua. **(Coming Soon)**
- 🎭 **NobarDrakor** — Platform khusus Drama Korea dan k-variety shows. **(Coming Soon)**
